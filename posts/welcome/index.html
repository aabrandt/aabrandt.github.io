Welcome to emacs.io

Welcome to my technical blog and portfolio site. I’m Andreas Aabrandt, an applied scientist and engineer specializing in production AI/ML systems.

What You’ll Find Here

This site serves as a space where I share insights, projects, and technical explorations at the intersection of:

  • AI & Machine Learning: LLM development, agent systems, fine-tuning techniques, and production deployment
  • Software Engineering: Building scalable systems, performance optimization, and architectural decisions
  • Aviation Technology: Flight planning software, avionics, and safety-critical systems
  • Research: Computational methods, optimization, and translating academic research into practical applications

Current Projects

I’m currently working on several exciting projects:

xWindAI (via Crosswind ApS)

A production LLM-based agent system featuring multi-step reasoning, tool orchestration, and RLHF fine-tuning. This system demonstrates how to build autonomous decision-making capabilities for complex enterprise scenarios while maintaining accuracy, efficiency, and cost-effectiveness.

Argos Flight Planning (via Crosswind ApS)

Real-time flight planning software leveraging GPU computing for performance co-simulation. This project combines my passion for aviation with high-performance computing to create tools that help pilots make better decisions.

Nexus (via Pionero ApS)

An intelligent data integration platform with ML-driven insights and anomaly detection. Built to solve the real-world problem of making sense of complex data integrations across enterprise systems.

Why “emacs.io”?

As a long-time Emacs user and advocate for powerful, extensible tools, this domain reflects my philosophy: combine deep technical knowledge with practical, customizable solutions. Whether it’s configuring Emacs or building production ML systems, the principles remain the same—understand the fundamentals, optimize relentlessly, and never settle for “good enough.”

  • Aviation software development and DO-178C compliance
